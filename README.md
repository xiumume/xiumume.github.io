# 随机点名工具
---
### 介绍
这是一个用于课堂上的一款小程序，这里我用的是传统的HTML+JS框架搭建，你可以部署到你的计算机上使用。
#### 部署服务环境
下载下来后，你若果会部署web服务器的话可以跳过直接看下一个环境。  
这里你可以使用xampp工具搭建一个web服务环境，你可以访问我的[展示网站](http://xiumume.github.io)看看是否一致，  
如果一致就可以下一步了。
#### 名单数据
你可以打开文件目录下的name.json文件看一下文件格式，你就可以将名字替换或者重新起草一份name.json的文件了。
```json
[
    {
        "name": "王文博"
    },
    {
        "name": "王仁良"
    },
    {
        "name": "范乐"
    },
    {
        "name": "柯昌利"
    }
]
```
你可以通过这样的一个Json文件来存储你等名单，考虑这样不太方便后期我会有改进。
