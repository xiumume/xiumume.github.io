# 朽木自雕网页随机点名工具
---
### 介绍
这是一个用于课堂上的一款小程序，这里我用的是传统的HTML+JS框架搭建，你可以部署到你的计算机上去使用它。
#### 部署服务环境
下载文件后，你如果会部署web服务器的话可以跳过直接看下一步。  
如果你是Windows的话，你可以使用xampp工具搭建一个web服务环境，Linux用户也可以使用Apache，具体教程可以参考[我的博客](https://bradghost.github.io)，按照我的博客搭建web环境完成后，你可以访问我的[展示网站](http://xiumume.github.io)看看是否一致，  
如果一致就可以下一步了，如果你在部署的过程中遇到难题也可以联系我。
#### 名单数据
你可以打开文件目录下的name.json文件看一下文件格式，你就可以将名字替换或者重新起草一份name.json的文件了。
```json
[
    {
        "name": "张三"
    },
    {
        "name": "李四"
    },
    {
        "name": "王二"
    },
    {
        "name": "麻子"
    }
]
```
你可以通过这样的一个Json文件来存储你的名单，不过考虑到这样不太方便后期我会有改进。
