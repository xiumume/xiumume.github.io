# 朽木自雕网页随机点名工具
---
### 介绍
这是一个用于课堂上的一款小程序，这里我用的是传统的HTML+JS框架搭建，你可以部署到你的计算机上去使用它。
#### 部署服务环境
下载文件后，你如果会部署web服务器的话可以跳过直接看下一步。  
如果你是Windows的话，我推荐使用xampp工具搭建web服务环境，Linux用户可以使用Apache，具体教程参考[我的博客](https://bradghost.github.io)，按照博客内容搭建web环境完成后，你可以访问我的[展示网站](http://xiumume.github.io)看看是否一致，一致就看下一步了，如果你在部署的过程中遇到难题也可以联系我。
#### 名单数据更改
你试着打开根文件目录下的name.json文件，看一下这个文件的格式，你就可以将名字替换或者重新起草一份name.json的文件了。  
如下是我重新起草的一份json文件，我们使用name.json替换掉根文件目录下原来的name.json即可。
```json
[
    {
        "name": "张三"
    },
    {
        "name": "李四"
    },
    {
        "name": "王二"
    },
    {
        "name": "麻子"
    }
]
```
你也可以试着起草这样的一份json文件来存储你的名单，不过考虑到这样并不太方便，我添加了一个可以辅助创建名单的网页，位于首页的“创建名单”按钮。
#### 完成部署
你是使用本地部署，访问127.0.0.1这个回送地址加端口号来访问你的工具，如果你是使用服务器部署，则输入你服务器的IP地址加端口号来访问。
